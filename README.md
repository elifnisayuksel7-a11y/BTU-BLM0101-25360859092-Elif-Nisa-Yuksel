# Proje Konusu: Veri Manipülasyonu ve Mantık Kapıları

**Proje Açıklaması:** Bu proje, Python programlama dili kullanılarak temel mantık kapılarını (AND, OR, NOT, XOR) simüle eden ve ayrıca A XOR (B AND C) ifadesinin doğruluk tablosunu oluşturan bir uygulamadır.

Program, kullanıcıdan ikili (binary) girişler (0 ve 1) alır, seçilen mantık kapısına göre işlemi gerçekleştirir ve sonucu ekrana yazdırır. Ardından üç değişkenli bir mantık ifadesi için tüm olası kombinasyonları hesaplayarak doğruluk tablosunu gösterir.

# Projenin Çalışma Mantığı

* **Projemiz iki ana bölümden oluşmaktadır.**
  # Birinci Bölüm
    * Projemizin ilk ana bölümünde kullanıcıdan 1 veya 0 girdileri input fonksiyonu kullanılarak alınacaktır. Kullanıcıdan alınan bu input bilgileri string olacağı için bu değerler integer değerlere dönüştürülecektir.
    * Kullanıcıdan 1 veya 0 değeri istediğimiz için girdiler kontrol edilerek istenilen şekilde değilse tekrar girdi alınacaktır.
    * Girdiler istenen şekilde sağlandıktan sonra kullanıcıdan işlem yapmak istediği temel mantık kapısını seçmesi istenecektir. Doğru seçim yapana kadar bu işlem while döngüsü ile tekrarlanacaktır.
    * Kullanıcının başlangıçta girdiği girdi değerleri seçtiği mantık kapısına göre işleme alınarak sonuç ekrana bastırılacaktır.
  # İkinci Bölüm
    * Projemizin ikinci bölümünde iç içe for döngüleri kullanılarak doğruluk tablosu oluşturulacaktır.
    * Bu doğruluk tablosu içerisinde A XOR (B AND C) ifadesinin değerleri gösterilecektir.
    * İç içe for döngüsü kullanılarak tüm ihtimaller değişkenlerin alacağı tüm değerlere göre ortaya çıkan mantık kapısı değeri ekrana yazdırlacaktır.
